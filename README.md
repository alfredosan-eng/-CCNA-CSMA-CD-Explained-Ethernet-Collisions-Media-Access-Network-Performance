# 📡 CCNA - CSMA/CD & Ethernet Media Access Control (Chapter 9)

## 📘 Overview
This repository contains study material for Ethernet media access control and CSMA/CD based on Cisco CCNA.

Ethernet uses CSMA/CD (Carrier Sense Multiple Access with Collision Detection) to manage how devices share the network medium and handle collisions in legacy shared networks.

---

## 🎯 Topics Covered
- Shared media communication
- CSMA/CD mechanism
- Collision detection and handling
- Backoff algorithm
- Collision domains
- Hubs vs switches
- Ethernet timing and latency
